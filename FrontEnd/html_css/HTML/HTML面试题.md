## 2.块状元素和&内联元素

### 块状元素
display: block/table; 有div,h1,h2,table,ul,ol,p等，主要特点是独占一行。可以设置width,height,margin和padding属性。

### 内联元素
display:inline/inline-block;有span，img，input，button。
inline-block和inline的区别在于inline-block拥有block的宽度和高度特性。
